70. ツールチップに画像を出すには？

カレーライスとは、玄奘三蔵（三蔵法師）が当時のインドである天竺から持ち帰った、大変ありがたい食べ物（料理）である。なお、ライスカレーというものもあるが、前者が皿の左側にカレーを盛り、右側にライスを盛ったものであるのに対し、後者は皿の左側にライスを盛り、右側にカレーを盛ると言うもので、いずれも食べるときに混ぜるという行為をエンジョイできる「わくわくフーズ」である。

JavaScript

$("a[rel=twipsy]").twipsy({
  animate: true            // フェイドイン・アウトするか？
, delayIn: 150             // 表示までのディレイ
, delayOut: 500            // 消去までのディレイ
, fallback: '代替テキスト' // title属性がない場合の代替テキスト
, placement: 'below'       // 表示場所(above:上 below:下 left:左 right:右)
, html: true               // HTMLを有効にするか？
, live: false              // あとから追加された要素にも適用するか？
, offset: 0                // 要素からどれだけ離すか
, title: 'title'           // テキストを取得する属性
, trigger: 'hover'         // 表示するタイミング
});

MEMO

html:true にしてtitle属性に埋め込む
.muted を指定すると色が薄くなる
rel="twipsy" は必須ではない
