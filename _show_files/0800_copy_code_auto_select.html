クリックして CTRL + c でコピーしてください

  <form class="form-stacked">
    <div class="clearfix">
      <label>生成コード</label>
      <div class="input">
        <textarea class="copy-code large" rows="4" style="overflow: hidden"><script type="text/javascript" src="http://ext.nicovideo.jp/thumb_watch/sm13401"></script><noscript><a href="http://www.nicovideo.jp/watch/sm13401">【ニコニコ動画】(ｽｰﾊﾟｰﾌﾟﾚｲ)(ﾖｰﾖｰ)worodl yo-yo- contest 2005 2A 1st - Shinji Saito</a></noscript></textarea>

      </div>
    </div>
  </form>

JavaScript

$(".copy-code").focus(function () {
  // Firefox だとこれだけで選択可能
  // $(this).select();

  // Chromeにも対応するなら setTimeout が必要になる
  var el = this;
  setTimeout(function () { $(el).select(); }, 0);
});

MEMO

$(".copy-code") にフォーカスされたら $(this).select() する
Firefoxだとそれだけで動くが、Chrome だと setTimeout が必要になる
