41. ボタンで実行→実行中を表す

HTML

<div class="container">
  <button id="main" data-loading-text="実行中..." class="btn btn-primary">実行</button>
</div>

JavaScript

$ ->
  $("#main").click ->
    btn = $(this)
    btn.button("loading")
    setTimeout ->
      btn.button("reset")
    , 1000

MEMO

ボタンに設定した data-loading-text は btn.loading("loading") で設定される
