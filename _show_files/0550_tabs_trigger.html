55. タブが変更されたときの検知方法

  • t1
  • t2
c1
c2

HTML

<div class="container">
  <ul class="tabs" data-tabs="tabs">
    <li class="active"><a href="#t1">t1</a></li>
    <li><a href="#t2">t2</a></li>
  </ul>
  <div class="tab-content">
    <div class="active" id="t1">c1</div>
    <div id="t2">c2</div>
  </div>
</div>

JavaScript

$(".tabs a").change(function(e){
  $(".container").append($(e.relatedTarget).attr("href") + " → " + $(e.target).attr("href") + "<br/>")
});

MEMO

フックはとくに用意されていない。jQueryの元の機能を使う
