HTML

<div class="container">
  <h3>1行用 + 閉じるボタン付き(x4色)</h3>
  <div class="alert-message info" data-alert="alert">
    <a class="close">&times;</a>
    <p><strong>Info: </strong>メッセージが1件届いています</p>
  </div>
  <div class="alert-message success" data-alert="alert">
    <a class="close">&times;</a>
    <p><strong>Success: </strong>メッセージが届いています</p>
  </div>
  <div class="alert-message warning" data-alert="alert">
    <a class="close">&times;</a>
    <p><strong>Warning: </strong>ニックネームを入力してください</p>
  </div>
  <div class="alert-message error" data-alert="alert">
    <a class="close">&times;</a>
    <p><strong>Error: </strong>エラーが発生しました</p>
  </div>
</div>

MEMO

閉じる機能をつける場合は bootstrap-alerts.js が必要
data-alert 属性があれば自動的に $.alert が反応する
