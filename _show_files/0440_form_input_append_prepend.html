JavaScript

$ ->
  $(".add-on :checkbox").on "click", ->
    if $(this).attr("checked")
       method = "addClass"
    else
       method = "removeClass"
    $(this).parents(".add-on")[method]("active")

MEMO

div.input input-prepend span.add-on input で前
前後につけることはできない
