77. ポップアップに閉じるボタンを入れるには？

起動

閉じるボタンは

自力で対応

閉じる

HTML

<div class="container">
  <a class="btn" data-controls-modal="main" data-backdrop="false" data-keyboard="true" href="#">起動</a>
  <div class="modal hide fade" id="main">
    <div class="modal-header">
      <h3>閉じるボタンは</h3>
    </div>
    <div class="modal-body">
      <p>自力で対応</p>
    </div>
    <div class="modal-footer">
      <a href="#" class="btn primary modal-close">閉じる</a>
    </div>
  </div>
</div>

JavaScript

$(".modal-close").click(function(){$(this).parents(".modal").modal("hide")})

MEMO

もっとシンプルに $(".modal-close").click(function(){$(".modal").modal("hide")}) でも良い
