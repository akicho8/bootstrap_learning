62. ポップアップダイアログのコントロール例

Aタグだけで起動(バックドロップ有効・ESC有効)


×

ヘッダー

本体

フッター

ボタン1 ボタン2

HTML

<div class="container">
  <a data-toggle="modal" href="#main">Aタグだけで起動(バックドロップ有効・ESC有効)</a>
  <hr/>
  <button id="c_kick1" class="btn">JSで表示(バックドロップ有効・ESC有効・表示)</button><br/>
  <button id="c_kick2" class="btn">JSで起動(内部で起動)</button><br/>
  <button id="c_toggle" class="btn">toggle</button>
  <button id="c_show" class="btn">show</button>
  <button id="c_hide" class="btn">hide</button>

  <div class="modal hide fade" id="main">
    <div class="modal-header">
      <a data-dismiss="modal" class="close">&times;</a>
      <h3>ヘッダー</h3>
    </div>
    <div class="modal-body">
      <p>本体</p>
    </div>
    <div class="modal-footer">
      <p>フッター</p>
      <a href="#" class="btn btn-primary">ボタン1</a>
      <a href="#" class="btn">ボタン2</a>
    </div>
  </div>
</div>

JavaScript

$ ->
  $("#c_kick1").click -> $("#main").modal({backdrop:true, keyboard:true, show:true})
  $("#c_kick2").click -> $("#main").modal()
  $("#c_toggle").click -> $("#main").modal("toggle")
  $("#c_show").click -> $("#main").modal("show")
  $("#c_hide").click -> $("#main").modal("hide")

MEMO

data-toggle   = modal はお約束
data-backdrop = true:他の部分をクリックするとダイアログを閉じる false:閉じない(背景もそのまま)
data-keyboard = true:ESCで閉じる
