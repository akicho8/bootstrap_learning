50. タブが変更されたときの検知方法

  • (t1)
  • (t2)
c1
c2

HTML

<div class="container">
  <ul class="nav nav-tabs">
    <li class="active"><a href="#t1" data-toggle="tab">(t1)</a></li>
    <li><a href="#t2" data-toggle="tab">(t2)</a></li>
  </ul>
  <div class="tab-content">
    <input type="text" id="current_tab" />
    <div class="tab-pane active" id="t1">c1</div>
    <div class="tab-pane"        id="t2">c2</div>
  </div>
  <hr/>
  <div id="console" class="well"></div>
</div>

JavaScript

$ ->
  $("a[data-toggle=tab]").on "shown", (e) ->
    # Hiddenタグなどに現在のタブIDを埋め込む例
    $("#current_tab").val($(e.target).attr("href").replace("#", ""))

  $("a[data-toggle=tab]").bind "show shown", (e) ->
    $("#console").prepend("" + e.timeStamp + " " + e.type + " " + $(e.relatedTarget).attr("href") + " → " + $(e.target).attr("href") + "<br/>")

MEMO

changeフックから前(relatedTarget)と後(target)が取得できる
