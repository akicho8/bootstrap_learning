73. クリックするだけで全選択するには？

クリックして CTRL + c でコピーしてください

HTML

<div class="container">
  <form class="form-vertical">
    <div class="control-group">
      <label class="control-label">生成コード</label>
      <div class="controls">
        <textarea class="copy-code input-large" rows="4" style="overflow: hidden"><script type="text/javascript" src="http://ext.nicovideo.jp/thumb_watch/sm13401"></script><noscript><a href="http://www.nicovideo.jp/watch/sm13401">【ニコニコ動画】(スーパープレイ)(ヨーヨー)worodl yo-yo- contest 2005 2A 1st - Shinji Saito</a></noscript></textarea>
        <p class="help-block">クリックして CTRL + c でコピーしてください</p>
      </div>
    </div>
  </form>
</div>

JavaScript

$ ->
  $(".copy-code").focus ->
    # Firefox だとこれだけで選択可能
    # $(this).select()

    # Chromeにも対応するなら setTimeout が必要になる
    setTimeout =>
      $(@).select()
    , 0

MEMO

$(".copy-code") にフォーカスされたら $(this).select() する
Firefoxだとそれだけで動くが、Chrome だと setTimeout が必要になる
